# 重点行业授信测算H5工具

## 项目简介

本项目是一款纯前端无后台依赖的"重点行业授信测算"H5工具，用于集成至微信服务号菜单功能。用户可以通过选择行业、填写相关数据，快速获取预估授信额度。

## 技术栈

- HTML5
- CSS3
- 原生JavaScript

## 功能特点

1. **三步式流程**：行业选择 → 数据填写 → 结果展示
2. **9个行业支持**：机械加工、粮食购销、粮食加工、粮食种植、苹果种植、果蔬仓储、餐饮、烟草、零售百货
3. **实时表单验证**：数字类型、范围限制、必填项检查
4. **响应式设计**：适配主流移动设备（屏幕尺寸4.7-6.7英寸）
5. **微信浏览器兼容**：支持微信内置浏览器，包括页面标题设置及微信分享功能基础配置
6. **纯前端实现**：所有数据仅存储在浏览器内存中，页面刷新或关闭后自动清除
7. **清晰的结果展示**：包含原始数据核对、测算公式说明及最终拟授信额度
8. **搜索able行业选择**：支持通过关键词搜索行业，提高用户体验
9. **行业专属测算逻辑**：每个行业都有独立的测算规则和准入条件
10. **美观的UI设计**：现代化的设计风格，良好的视觉层次和交互体验

## 部署方法

### 本地开发
1. 克隆仓库：`git clone https://github.com/Fan-YuP/key-industry-credit.git`
2. 进入项目目录：`cd key-industry-credit`
3. 启动本地服务器：`python -m http.server 8000`（或使用其他HTTP服务器）
4. 在浏览器中访问：`http://localhost:8000`

### 生产部署
1. 将所有文件上传至静态托管平台（如Gitee Pages、GitHub Pages、腾讯云COS、阿里云OSS等）
2. 生成公网可访问的H5链接
3. 将链接配置到微信服务号菜单中

## 行业测算逻辑

### 1. 机械加工行业

#### 必填字段
- 年经营流水（单位：万元）
- 近一年销售收入（单位：万元）
- 上一年销售收入（单位：万元）
- 家庭资产（单位：万元）
- 总负债（单位：万元）
- 家庭负债（除房贷、车贷、抵押类贷款外，其他信用、保证负债，单位：万元）
- 经营年限（单位：年，精确到0.5年，且必须满1年）
- 准入条件验证：
  1. 企业是否不存在重大行政处罚记录，且不存在重大涉诉涉案情况

#### 测算公式
1. 单户最高授信额度 = 年经营流水 × 20%，最高200万元
2. 我行拟授信额度 = 单户最高授信额度 - 家庭负债，最低0万元

### 5. 苹果种植行业

- 家庭负债（除房贷、车贷、抵押类贷款外，其他信用、保证负债，单位：万元）
- 经营年限（单位：年，精确到0.5年，且必须满3年）
- 果树种植亩数（单位：亩，不低于10亩）
- 准入条件验证：
  1. 是否具备3年及以上种植经验，果树管理良好且具备固定住所或稳定经营场所

#### 测算公式
1. 单户最高授信额度 = 果树种植亩数 × 5600元/亩
2. 我行拟授信额度 = 单户最高授信额度 - 家庭负债，最低0万元

### 7. 餐饮行业

#### 必填字段
- 年经营流水（单位：万元）
- 近一年销售收入（单位：万元）
- 上一年销售收入（单位：万元）
- 投资总成本（单位：万元）
- 家庭资产（单位：万元）
- 总负债（单位：万元）
- 家庭负债（除房贷、车贷、抵押类贷款外，其他信用、保证负债，单位：万元）
- 经营年限（单位：年，精确到0.5年，且必须满1年）
- 贷款类型（抵押类、保证类、信用类）
- 准入条件验证：
  1. 是否持续经营1年以上且无行政处罚、诉讼记录及食品安全事故

#### 测算公式
1. 抵押类单户最高授信额度 = 年经营流水 × 60%
2. 保证类单户最高授信额度 = 年经营流水 × 45%，最高200万元
3. 信用类单户最高授信额度 = 年经营流水 × 20%，最高200万元
4. 近一年销售收入较上一年销售收入下降的，单户最高授信额度 × 90%
5. 我行拟授信额度 = 单户最高授信额度 - 家庭负债，最低0万元

### 8. 烟草行业

#### 必填字段
- 年经营流水（单位：万元）
- 近一年销售收入（单位：万元）
- 上一年销售收入（单位：万元）
- 家庭资产（单位：万元）
- 总负债（单位：万元）
- 家庭负债（除房贷、车贷、抵押类贷款外，其他信用、保证负债，单位：万元）
- 经营年限（单位：年，精确到0.5年，且必须满1年）
- 近12个月订烟进货额（单位：万元）
- 往年订烟量最高月份进货额（单位：万元）
- 近12个月零售类经营收入（不含烟草零售收入，单位：万元）
- 准入条件验证：
  1. 企业是否不存在重大行政处罚记录和重大涉诉涉案情况
  2. 是否取得烟草专卖局颁发的《烟草专卖零售许可证》

#### 测算公式
1. 额度一 = max(近12个月订烟进货额 × 50%, 往年订烟最高月进货额 × 1.5)
2. 额度二 = 近12个月零售类经营收入 × 15%
3. 总授信额度 = 额度一 + 额度二
4. 信用类贷款额度 = min(总授信额度, 200万元)
5. 需追加担保额度 = max(总授信额度 - 200万元, 0)

### 9. 零售百货行业

#### 必填字段
- 年经营流水（单位：万元）
- 近一年销售收入（单位：万元）
- 上一年销售收入（单位：万元）
- 家庭资产（单位：万元）
- 总负债（单位：万元）
- 家庭负债（除房贷、车贷、抵押类贷款外，其他信用、保证负债，单位：万元）
- 经营年限（单位：年，精确到0.5年，且必须满1年）
- 准入条件验证：
  1. 是否有固定经营场所、营业执照，持续经营1年以上，近1年无行政处罚或诉讼记录
  2. 近2年是否无严重逾期记录（如连续3个月以上逾期）

#### 测算公式
1. 单户最高授信额度 = 年经营流水 × 30%，最高80万元
2. 我行拟授信额度 = 单户最高授信额度 - 家庭负债，最低0万元

## 使用说明

1. 打开H5页面，进入"行业选择"步骤
2. 在搜索框中输入行业关键词或直接从下拉列表中选择行业
3. 点击"下一步"进入"数据填写"步骤
4. 填写相关表单数据，确保所有必填字段都已填写且符合验证规则
5. 点击"立即测算"按钮，系统将根据您选择的行业和填写的数据进行测算
6. 查看"结果展示"页面，了解预估授信额度及测算详情
7. 如需重新测算，点击"重新测算"按钮返回行业选择页面

## 扩展说明

本项目采用模块化设计，便于后续扩展其他行业的测算功能。扩展步骤如下：

1. **添加行业配置**：在`calculator.js`的`industryConfigs`对象中添加新行业的配置信息
2. **实现测算逻辑**：在`calculator.js`中添加新行业的测算方法
3. **添加表单字段**：在`index.html`中添加新行业的表单结构
4. **实现表单验证**：在`validation.js`中添加新行业的验证规则
5. **添加结果展示**：在`calculator.js`的`updateResultDetails`方法中添加新行业的结果展示逻辑
6. **更新表单切换**：确保`script.js`能够正确切换到新行业的表单
